# Nexus Intelligence 2026 – Scalable E-Commerce Predictive Analytics Platform

## Project Overview

Nexus Intelligence 2026 is an end-to-end Data Science and Machine Learning platform developed on 1M+ E-Commerce transactions.

The platform provides:

* Fraud Detection
* Customer Segmentation
* Sales Forecasting
* Return Prediction
* Executive KPI Analytics
* Interactive Streamlit Dashboard

## Tech Stack

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-Learn
* Streamlit
* Machine Learning
* Data Analytics

## Machine Learning Modules

### Fraud Detection

Random Forest Classifier for identifying high-risk transactions.

### Customer Segmentation

K-Means Clustering for customer grouping.

### Sales Forecasting

Predictive model for future sales estimation.

### Return Prediction

Customer return behavior prediction system.

## Dashboard Features

* Revenue KPIs
* Profit KPIs
* Customer Analytics
* Fraud Monitoring
* Country-wise Revenue Analysis
* Category Performance Analysis

Dataset contains 1M+ E-Commerce transactions.
